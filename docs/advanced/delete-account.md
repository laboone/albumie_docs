# アカウントの削除
アカウントの削除は以下の操作で実行できます。

1. ページ右上のアカウントアイコンをクリックし、表示されるプルダウンメニューから「アカウント」をクリックします。

2. アカウントページで「アカウントを削除」ボタンをクリックします。

自分以外に管理者アカウントがいない場合は、アップロードした写真や招待済みアカウントを含むすべてのデータが削除されます。  
自分以外に管理者アカウントがいる場合は、アカウント削除を実行したアカウント情報のみ削除されます。

:::caution
削除したデータは元には戻せません。
:::
