# Can face recognition work if faces change over time?
If facial appearance changes significantly (for example, lower grades vs. upper grades),
accuracy for recognizing the same person may decrease.
To improve accuracy, use the [Collection feature](/advanced/create-collection)
and manage lower-grade and upper-grade photos separately.

Recommended steps:
1. Create a collection for lower-grade years
2. Create a collection for upper-grade years
3. Register people and analyze photos in each collection

By separating collections and registering people per collection,
you can achieve more accurate face recognition.
This also helps you quickly compare photo counts per person between lower and upper grades,
which prevents selecting too many photos from only one period and helps keep balance.
