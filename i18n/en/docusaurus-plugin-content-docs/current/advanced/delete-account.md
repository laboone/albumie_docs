# Delete Account
You can delete your account using the following steps:

1. Click the account icon in the top-right corner of the page,
   then click **Account** in the drop-down menu.

2. On the account page, click **Delete Account**.

If you are the only administrator,
all data will be deleted, including uploaded photos and invited accounts.  
If there is another administrator account,
only the account that performed the deletion will be deleted.

:::caution
Deleted data cannot be restored.
:::
