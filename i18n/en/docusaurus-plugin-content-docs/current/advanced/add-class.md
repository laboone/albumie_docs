# Set Classes
You can assign classes to people.
After setting classes, you can check photos by class on the album **People View** page.

Steps:

1. Click **People** in the left menu.
2. Click **Edit Classes** in the top-right area.
   This switches to people edit mode.


3. Click people to select them.
   Selected photos are highlighted in blue.
   Click again to deselect.

4. Click **Set Class for Selected People** in the top-right area.
   A class input popup appears. Enter the class and click **Set**.
   This assigns the class to selected people.
   To remove a class, leave the class field empty and click **Set**.

5. After finishing, click **Done** in the top-right area.

After classes are set, a class dropdown appears and you can filter people by class.
On the album **People View** page, the class dropdown is also available.
