# Manage Photos by Collection
## Create a Collection
If you need to keep photos separate (for example by school or grade),
you can manage them with the **Collection** feature.
When you create an account, a collection named **Default** is created automatically,
and uploaded photos are stored there.
If you want to separate photos, create a new collection.

Steps to create a new collection:

1. In the collection dropdown menu, click **Create Collection**.
   The **Create Collection** popup appears. Enter a collection name and click **Create**.

2. The new collection is created.
   To switch back to another collection, use the collection dropdown menu.

:::tip
You can create up to 10 collections.
:::

## Delete a Collection
You can delete created collections with these steps:

1. Click the settings icon (gear icon) in the header.


2. On the settings page, click **View** in the collection management section.
   The collection list page appears.


3. Click **Delete** next to the collection you want to remove.
   A confirmation popup appears. Click **Delete** to remove it.

:::tip
If there is only one collection, it cannot be deleted.
Create another empty collection first, then delete the target one.
:::

:::caution
Deleting a collection deletes all photos, albums, and other data in that collection.
Deleted data cannot be restored.
:::
