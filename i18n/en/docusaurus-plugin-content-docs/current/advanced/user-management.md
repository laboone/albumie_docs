# Invite Users
By inviting users to Albumie, multiple people can collaborate on photo selection.

Steps:

1. Click the settings icon (gear icon) in the header.


2. On the settings page, click **View** in the user management section.
   The user management page appears.

3. Enter the email address of the person you want to invite,
   and select a permission level.
   If you choose **Member**, select collections that the member can access.
   For collection details, see [Manage Photos by Collection](create-collection.md).

:::info About permissions
**Administrator** has the same permissions as the inviter and can access all collections.
**Member** can access only permitted collections.
Members also cannot open the settings page.
:::

4. Click **Send**.
   An invitation email with a temporary password is sent to the entered email address.
   Ask the invited person to log in to Albumie.

:::info
If an account already exists for that email address, you cannot invite it.
Please invite using a different email address.
:::

:::tip About deleting invited accounts
You can delete invited accounts from the user list in the lower section of user management.
Once deleted, that account can no longer log in to Albumie.
:::
