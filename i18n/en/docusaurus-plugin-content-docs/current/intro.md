# Introduction
This document is the online manual for [Albumie](https://albumie.app/en).

Albumie is a graduation album creation support service with AI-powered face recognition.
You can check photos and photo counts for each student and select pictures so that each student appears in a balanced way.
If this is your first time, we recommend going through [Basic Operations](/tutorial/intro), from uploading photos to selecting photos.
