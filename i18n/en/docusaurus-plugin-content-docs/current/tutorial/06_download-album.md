# Download Album
For photos added to an album, you can download a file name list and the original uploaded files.
Click the **︙** button in the top-right corner and choose:
- **Download File Name List** to download the list
- **Download Files** to download files


:::info
When downloading files, your browser may show a popup asking for permission to download multiple files.
Please allow it.
:::

That completes the basic operations.

On the [Advanced Operations](/advanced/create-collection.md) page,
you can learn photo management methods and how to invite other users.
Please check it as needed.
